# 2.4 GHz Microstrip Patch Antenna Design using HFSS

## Overview
This project presents the design and simulation of a microstrip patch antenna using ANSYS HFSS.

## Software
- ANSYS HFSS 2025 R2 Student

## Antenna Specifications

| Parameter | Value |
|------------|------------|
| Antenna Type | Microstrip Patch |
| Operating Frequency | 2.285 GHz |
| Return Loss (S11) | -31.79 dB |
| VSWR | 1.05 |
| Maximum Gain | 1.56 dBi |
